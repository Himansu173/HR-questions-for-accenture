13. What is your role in the group project

In our group project, my primary role was developing the vendor account verification process. I implemented features such as real-time face selfie for biometric verification and phone OTP verification, using Firebase Storage and Firebase Phone Auth. This involved setting up user-friendly authentication processes. My work required collaboration with the team to integrate these features smoothly into the application.

14. What challenges did you face in the project

In our project, Evehunt, we encountered both technical and non-technical challenges. A significant non-technical challenge was compatibility issues when merging the different components developed by our team members. Each of us had slightly different coding styles, which led to conflicts during integration. To address this, we adopted a standardized coding structure, and we shared our code on GitHub regularly to keep everyone updated and aligned. The main technical challenge was storing images using GraphQL, which proved complex and inefficient for handling media files. After assessing various options, we decided to store images in Firebase instead. Firebase offered a more flexible and scalable solution for our media storage needs.
